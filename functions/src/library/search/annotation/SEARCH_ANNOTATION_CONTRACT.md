# Search Annotation Contract

Status: LOCKED (v1.0)

Purpose:
Search Annotation enriches Search results with contextual,
explainable, non-authoritative signals.

Hard Rules:
- MUST NOT change ranking
- MUST NOT reorder results
- MUST NOT filter or inject items
- MUST NOT fetch data
- MUST NOT influence Search decisions

Search remains authoritative at all times.
Annotations are optional, dismissible, and non-binding.
